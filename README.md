Nexus is a Python-based AI voice assistant capable of performing system-level tasks, handling voice and text commands, and interacting through a web-based UI.

## Features
- Wake word activation ("Nexus")
- Voice and text command support (via UI)
- Opens applications
- Plays music
- Set reminders
- Takes screenshots
- Shutdown / Restart / Sleep system
- Image generation
- Image analysis
- Automates system-level tasks

## Tech Stack
- Python (Core Logic)
- Flask (Backend)
- HTML, CSS, JavaScript (Frontend UI)
- APIs (for AI functionalities)

## How It Works
- Backend built using Flask handles requests and system operations  
- Frontend UI allows interaction via text and voice  
- AI processes commands and executes tasks in real-time
